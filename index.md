# 生物信息学 (Bioinformatics) 学习记录

## 1. 课堂笔记：算法 (Algorithm) 与 模型 (Model) 的辨析

在计算生物学领域，明确“模型”与“算法”的边界是理解数据分析框架的基础：

* **模型 (Model)**：对现实生物学系统或数据生成机制的数学与逻辑抽象，旨在描述事物的内在关系或数据的分布特征。例如，用于描述序列概率特征的隐马尔可夫模型 (HMM)，或用于预测分子属性的深度神经网络。
* **算法 (Algorithm)**：为解决特定问题或求解模型而设计的一系列明确、有限的计算步骤与指令集合。例如，用于局部序列比对的 Smith-Waterman 算法，或用于优化神经网络权重的反向传播算法。

**总结**：在实际研究中，我们通常通过执行**算法**来训练和求解**模型**，进而利用该**模型**对复杂的生物学数据进行预测与物理解释。

## 2. 本学期生物信息学学习计划

依据本学期的教学大纲，我的学习计划主要分为以下四个阶段，逐步实现从基础计算工具到前沿交叉应用的过渡：

### 阶段一：编程基础与工作流配置 (Week 1-4)
* 熟练掌握 Linux 基础命令与 Bash 脚本编程。
* 独立配置 Docker 运行环境，并建立基于 Git/GitHub 的版本控制工作流，为后续大规模数据分析奠定工程基础。

### 阶段二：组学分析与 R 语言实践 (Week 5-12)
* 全面掌握 R 语言基础及数据可视化方法。
* 结合统计学理论，完成从 RNA-seq 测序数据比对、差异表达分析，到 GO/KEGG 功能富集分析的标准流程；并尝试扩展处理 ChIP-seq、CLIP-seq 等多维组学数据。

### 阶段三：机器学习与前沿探索 (Week 13-16)
* 掌握机器学习的基础理论与经典模型架构。
* 借助 Python 探索计算方法在生物医学中的应用，特别是结合 AI 辅助药物发现 (AIDD) 等前沿方向，尝试阅读并复现经典文献的计算过程。

### 阶段四：期末项目与综合应用
* 在期末 Final Project 中整合所学计算工具。
* 尝试将多组学数据分析（如共表达网络分析）与自动化脚本，实际应用于具体的课题设计中（如辅助合成生物学基因元件的筛选与优化），将理论知识转化为解决生命科学问题的有效手段。
